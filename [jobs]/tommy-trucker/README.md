# 🚚 TOMMY TRUCKER v2.0 - BOX TRUCK VERSION

## ✅ CẬP NHẬT CHÍNH:
- ✔️ CHỈ XE TẢI THÙNG (Mule, Benson, Boxville...)
- ✔️ KIỂM TRA TRUNK (cửa thùng sau - door 5)
- ✔️ SQL query KHÔNG kiểm tra state=1

## 📦 CÀI ĐẶT:
1. Import `truck_driver.sql`
2. Copy folder vào `resources/[qb]/`
3. Add `ensure tommy-trucker` vào server.cfg
4. Restart server

## 🚗 XE WHITELIST:
- Mule, Mule2, Mule3, Mule4, Mule5
- Benson, pounder2, pounder2
- Youga, Speedo, Rumpo, Burrito (series)

## 🎮 GAMEPLAY:
1. `/car mule3` - Spawn xe
2. Mở **2 CỬA SAU** xe (door 2 & 3) - Interaction Menu
3. Đi NPC → Đăng ký xe → Nhận đơn
4. Đến pickup → **E** bốc hàng
5. Sau xe → **E** xếp vào thùng
6. Lặp lại đến đủ kg
7. Đến dropoff → Mở 2 cửa sau → **E** lấy hàng
8. **E** giao hàng → Nhận thưởng

## 🔧 NOTES:
- Door 2 = Cửa sau trái, Door 3 = Cửa sau phải
- Phải mở CẢ 2 cửa trước khi bốc/trả hàng
- Police có thể kiểm tra hàng cấm (qb-target)
