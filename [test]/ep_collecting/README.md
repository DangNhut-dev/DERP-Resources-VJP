# EP Collector Zones
Collector Zones dynamically manages in-game items within specified zones, handling their spawning, placement, and removal based on player interaction with these zones. The script defines zones where items can appear, randomly places items within these zones, and ensures items are properly interacted with through custom events. It also cleans up items when the resource stops, ensuring no leftover items remain in the game world. This functionality is particularly useful for creating immersive, interactive environments where players can find and interact with objects that appear differently each time they enter a zone, enhancing the gameplay experience with dynamic content.

## Features
- **Dynamic Item Placement**: Items are randomly placed within specified zones, allowing for a unique experience each time a player enters a zone.
- **Custom Events**: Custom events are triggered when players interact with items, allowing for unique interactions and gameplay mechanics.
- **Zone Management**: Zones are defined and managed through the script, allowing for easy configuration and customization.
- **Automatic Cleanup**: Items are automatically removed when the resource stops, ensuring no leftover items remain in the game world.

## What's Included
- **Client Script**: The client script handles the rendering of items and interactions with the server.
- **Collectable custom props**
75 different models to be used in FiveM. Each plant includes 3-4 different growing stages.
This resource is intended to be used by experienced users and with farming scripts or standalone.
No customer support for using models themselves is provided.
